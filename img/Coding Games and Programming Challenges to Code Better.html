 Goal

You are given an array of N integers and a list of M inclusive ranges a, b.

For each range, print the sum of the values in the array with an index between a and b inclusive.
Input
Line 1: the number N of integers in the list.
Line 2: N space-separated integers from -100 to 100.
Line 3: the number M of ranges.
M next lines: two space-separated integers startRange and endRange. The first element of the list is at index 0.
Output
M lines: for each range, the sum of the values in the array of N integers with an index between a and b inclusive.
Constraints
0 < N < 100
0 < M < 100
